# ExoMind DevLog

ExoMind 项目开发日报公开归档。

**在线阅读**: https://exomind-team.github.io/exomind-devlog/

## 关于

每日开发报告由 [ExoMind](https://github.com/exomind-team/exomind) 项目的 dev-daily Agent 自动生成，涵盖：

- Issue / PR 全景数据
- 主线追踪与进度
- 天气预报与建议行动
- 代码 vs GitHub 真相表

## 发布

报告通过主仓库的发布脚本推送：

```bash
# 在 exomind 主仓库中
bun run devlog:publish
```
